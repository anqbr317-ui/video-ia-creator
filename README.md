# 🎬 Video IA Creator

Video IA Creator est une plateforme web permettant de générer rapidement des prompts professionnels pour créer des vidéos avec l'intelligence artificielle.

## 🚀 Fonctionnalités

- Génération de prompts vidéo IA
- Compatible TikTok, Instagram Reels et YouTube Shorts
- Choix du style visuel
- Choix du format vidéo
- Choix de la durée
- Ambiance sonore personnalisable
- Interface responsive mobile et ordinateur

## 🤖 Compatibilité

Les prompts générés peuvent être utilisés avec :

- Runway AI
- Kling AI
- Pika
- Hailuo AI
- Luma Dream Machine
- Canva AI
- Autres générateurs vidéo IA

## 💻 Installation

Clonez le dépôt :

```bash
git clone https://github.com/anqbr317-ui/video-ia-creator.git
```

Accédez au dossier :

```bash
cd video-ia-creator
```

Ouvrez simplement le fichier :

```bash
index.html
```

dans votre navigateur.

## 🌐 Déploiement GitHub Pages

1. Ouvrez votre dépôt GitHub.
2. Allez dans **Settings** → **Pages**.
3. Sélectionnez :
   - Source : **Deploy from a branch**
   - Branch : **main**
   - Folder : **/root**
4. Cliquez sur **Save**.

Votre site sera disponible à :

```text
https://anqbr317-ui.github.io/video-ia-creator/
```
